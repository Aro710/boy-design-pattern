1 Windows操作系统中的应用程序快捷方式是( )模式的应用实例。  
A. 代理 (Proxy) B. 组合 (Composite)  
C. 装饰 (Decorator) D. 外观 (Facade)

> 答：A （快捷方式做代理，客户无需关系具体实现）
